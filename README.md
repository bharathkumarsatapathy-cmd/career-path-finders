# 📌 Career Path Finder 🚀

Career Path Finder is a web application that helps students and job seekers identify suitable career opportunities based on their qualifications, interests, and preferred job sector.

🎯 Features
🔐 User Authentication (Login & Signup)
💾 Database Integration using SQLite
🎯 Personalized Job Recommendations
🧠 Interest-Based Career Suggestions
🎨 Clean UI with CSS and animations
🌐 Deployable web application

🛠️ Tech Stack
1. Frontend: HTML, CSS, Bootstrap
2. Backend: Python (Flask)
3. Database: SQLite
4. Deployment: Render

⚙️ How It Works

User signs up and logs in

Enters qualification, sector, and interests

System processes input using logic

Displays suitable career options
